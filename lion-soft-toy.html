How to personalise at checkout

Checkout will ask for three lines. Please keep each line to a maximum of 20 characters.

  • Line 1: your main message (printed in your chosen accent colour)
  • Line 2: name (printed in Grey by default)
  • Line 3: your style choices (Font | Accent colour | Optional icon code | Any notes)

Example (recommended two-tone):
Line 1: Missing you!
Line 2: Charlie
Line 3: Playful - Blue - C3

Defaults:
• We use a two-tone finish (Accent colour + Grey) for a calmer, premium look.
• Icons print in matching accent colour unless N1, N2, N3, N4, N5, N6 is chosen.
If you want everything in one colour, add: Single colour on Line 3.

Optional extras such as a robe or hi-vis vest can be added at checkout. Any personalisation on these items will automatically match the name and icon (if chosen) from your bear.

Choose your design guides

Use the cards below to decide your font, colour and optional icon code, then enter them on Line 3 at checkout.

Fonts

Choose: Playful or Soft rounded or Handwritten or Script.

Colours

Choose: Pink, Blue, Green, Red, or Grey.

Icons (optional)

If you want an icon, choose one icon code and add it to Line 3. If you do not want an icon, just leave it off.

Name icons

Everyday icons (for example: N1, N4, N6).

Celebration icons

Birthday and celebration icons (for example: C1, C4, C11).

Newborn icons

Baby-themed icons (for example: B1, B5, B7).

Memorial icons

Gentle memorial icons (for example: M1, M5, M7).
