How to personalise at checkout

Checkout will ask for three lines. You can use them in any order.
For the best print size and clarity, we recommend keeping each line to around 20 characters. Longer messages may be resized to fit the available print area.

  • Line 1: your text (styling notes in brackets)
  • Line 2: name or photo request (styling notes in brackets)
  • Line 3: optional memorial icon (styling notes in brackets)

Example (name + icon):
Line 1: Forever loved (script/grey)
Line 2: Charlie (handwritten/grey)
Line 3: M1 (icon-grey)

Example (photo + optional icon):
Line 1: In loving memory (handwritten/grey)
Line 2: PHOTO (email to follow)
Line 3: M2 (icon-grey)

Ordering more than one?

If you’re buying multiple bears, please list each bear’s details clearly in the personalisation box (for example: Bear 1 – Charlie / Bear 2 – Amelia).
