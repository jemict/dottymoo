How to personalise at checkout

Checkout will ask for name, icon code and accent colour. For the best print size and clarity, we recommend names to around 12 characters. Longer names may be resized to fit the available print area.

  • Name: Name to be printed
  • Icon Code: Enter a code (A, B, C, D, E, F or G)
  • Accent Colour: Used for the top message and matching icon

Example:
Name: Charlie
Icon Code: D
Accent Colour: Blue

Your text will be printed exactly as entered.
Please check spelling, spacing, and capitalisation carefully before checkout.

Ordering more than one?

If you’re buying multiple bears, please list each bear’s details clearly in the personalisation box (for example: Bear 1 – Charlie / Bear 2 – Amelia).
