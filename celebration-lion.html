What this is

The Celebration Lion Soft Toy is a confident keepsake gift designed for happy moments. Your chosen message and name are printed on the T-shirt, with optional colour and icon choices.

Choose your design

First, take a look at the available fonts, colours and optional icons below. You’ll then be shown exactly how to enter your choices at checkout.

Fonts

Choose: Playful, Soft rounded, Handwritten, or Script.

Accent colours

Choose: Pink, Blue, Green, Red, or Grey.

Icons (optional)

If you would like an icon, choose one icon code and include it with your style choices.

Name icons

Everyday icons (for example: N1, N4, N6).

Celebration icons

Birthday and celebration icons (for example: C1, C4, C11).

Newborn icons

Baby-themed icons (for example: B1, B5, B7).

How to personalise at checkout

At checkout, you will be asked for three pieces of information.

  • Celebration message (for example: Happy Birthday)
  • Name
  • Style (Font – Colour – Optional icon code – Any notes)

Example:
Celebration message: Happy Birthday
Name: Charlie
Style: Playful – Blue – C8

Defaults:
• We use a two-tone finish (accent colour + grey) for a balanced look.
• Icons print in the accent colour unless N1–N6 is chosen.
If you want everything in one colour, add: Single colour.
