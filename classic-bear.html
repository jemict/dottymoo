How to personalise at checkout

Checkout will ask for three lines. Please keep each line to a maximum of 20 characters.

  • Line 1: your main message
  • Line 2: your design choices (Font - Colour - Optional icon code)
  • Line 3: name (or an extra short line if you prefer)

Example:
Line 1: Missing you!
Line 2: Playful - Blue - N1
Line 3: Charlie

Choose your design guides

Use the cards below to decide your font, colour and optional icon code, then enter them on Line 2 at checkout.

Fonts

Choose: Playful or Soft rounded or Handwritten or Script.

Colours

Choose: Pink, Blue, Green, Red, or Grey.

Icons (optional)

If you want an icon, choose one icon code and add it to Line 2. If you do not want an icon, just leave it off.

Name icons

Everyday icons (for example: N1, N4, N6).

Celebration icons

Birthday and celebration icons (for example: C1, C4, C11).

Newborn icons

Baby-themed icons (for example: B1, B5, B7).

Memorial icons

Gentle memorial icons (for example: M1, M5, M7).
